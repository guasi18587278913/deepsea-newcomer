# 广场重做:微信群聊版 · 设计稿

- 日期:2026-06-05
- 状态:已与瓜斯逐点确认(brainstorming),进入实现
- 对象文件:`discuss.html`(原"微博式动态流"广场 → 重做成微信群聊)

## 背景

深海圈常态化运营、取消微信群,要把群聊互动搬进网页。课程交付 PRD(`prd-course-delivery-upgrade` v0.5)把"社区交流改版"划给虫二单独立项;会议纪要(妙记 `obcnz26n6v5h1m5ra2b1xcbl`)定方向:交流区拆「广场(聊天流)+ 精华帖(报纸式)」。**本次只做广场,做成微信群聊的样子。**

## 已定决策(brainstorming 逐点确认)

1. **形态**:多频道(微信 PC 版 —— 左会话/频道列表 + 右聊天窗)
2. **频道清单**:沿用现有 7 个热门话题 —— 大厅 / 今日上线 / 踩坑求助 / 产品变现 / 产品出海 / 开发者日常 / 千刀专属🔒
3. **配色**:微信的形 + 深海 teal —— 浅瓷白底 `#f5fafa`、别人白气泡、自己 teal 气泡 `#0d9488`、主色 teal
4. **布局**:两栏 —— 左频道列表 + 右聊天窗占满;在线成员点顶部「👥」展开;继续学习做成置顶**群公告条**
5. **消息形态**:微信气泡 + 留深海魂 —— 昵称带 Lv、报喜/求助角标、气泡挂可点课程关联(跳 learn)、学习成就做成系统灰条;互动 = 表情回应 / 引用回复 / @(砍掉点赞数 / 评论区 / 收藏 / 浏览 / 话题标签)

## 页面结构

- **顶栏**(全站复用):深海圈 | 交流区·学习中心·作品墙·成员 + 在线数 + 头像
- **左栏**(频道列表 ~240px,微信会话列表样式):7 频道(名 + 最后消息预览 + 未读红点),底部本人身份卡(瓜哥 Lv.2)
- **右栏**(聊天窗):
  - 顶部条:`#频道名` + 👥人数 + ⋯
  - 群公告条(置顶):📌 继续学习 18/77 →(= 原"继续学习"卡)
  - 消息流:气泡(别人左白 / 自己右 teal)+ 系统灰条
  - 输入框:文字 + 😀 + 📷 + @ + 发送
- **成员面板**:点👥展开(= 原"此刻在线":N 人在线 / 零基础数 / 成员列表)
- **刘小排浮窗**:右下复用(`assistant.js`)

## 消息气泡(留深海魂)

- 别人:头像 + 昵称(Lv)+(🎉报喜 / 🆘求助 角标)+ 白气泡(文字 / 图片)+ 课程关联标签(点跳 `learn.html?code=`)+ 时间
- 自己:右对齐 teal 气泡
- 互动:hover 出表情回应(👍🎉)、引用回复、@

## 系统灰条(学习 × 群聊串联)

群里穿插灰色系统消息:
- 🎉 XX 完成了 A3·做出第一个网站
- 🚀 XX 的产品上线了
- 👋 XX 加入了 #频道
- 📝 XX 提交了阶段作业

## 数据与持久化(纯 mock)

- 每频道预填历史消息数组 `MESSAGES[channel]`(发言人 / Lv / 内容 / 时间 / 课程 / 类型 / 表情数)
- 自己发的存 localStorage `ds_chat_<channel>`,刷新不丢
- 在线人数 / 成员 mock;大厅可选"假实时"偶尔冒新消息
- 千刀身份:localStorage `ds_is_qiandao` + demo 切换器模拟

## 千刀专属频道

- 左栏带🔒;点击非千刀 → 提示"千刀专属,审核后可进";千刀 → 可进
